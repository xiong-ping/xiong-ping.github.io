在Jupyer Notebook里安装包

Dec 09, 2019
1 minute read

一个方便的小技巧，可以很快的安装缺失的包。

可以使用pip或conda来安装包，注意如果环境不同包是不能通用的，在Windows下启动Anaconda Prompt的时候要注意标题后面括号内的环境是不是所需要的。

以numpy为例，使用pip：

import sys
!{sys.executable} -m pip install numpy

使用conda：

import sys
!conda install --yes --prefix {sys.prefix} numpy
