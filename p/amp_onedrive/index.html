AMPlibraryagent异常访问Onedrive下载

Apr 10, 2021
1 minute read

先放方法：用任务监视器定位到AMPlibraryagent的路径，然后使用OneDrive的“应用管理”，停止响应AMPlibraryagent的请求。

今天在正常使用MacBook时，偶然发现OneDrive经常会异常同步，反复释放空间无果。查看OneDrive的日志发现，是一个名叫AMPlibraryagent的应用请求的。查询得知，AMPLibraryAgent（The media library agent for Music.app and TV.app）是Apple Music和Apple TV的相关进程，属于系统进程，从它下手的可能性不大。

尝试很多方法后，终于发现可以通过onedrive设置里面的“管理应用”来停止响应特定应用的请求🔗。于是有了如下步骤：

  1. 任务监视器定位到AMPlibraryagent的路径。
  2. Onedrive > 首选项 > 应用管理 > +
  3. 弹出的Finder里 Cmd + Shift + G, 输入AMPlibraryagent的路径，选择它和AMPArtworkAgent，停止OneDrive对他们的响应。
  4. ok
